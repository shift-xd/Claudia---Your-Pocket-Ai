 #### Claudia uses the following parts 

| Part Name         | Quantity | Description / Specs                  | Notes |
|-------------------|-----------|-------------------------------------|--------|
| ESP32 Dev Board   | 1         | WiFi + Bluetooth microcontroller    | Main control board |
| MPU6050           | 1         | 6-axis accelerometer + gyroscope    | Motion detection |
| INMP441           | 1         | I2S digital microphone              | Audio input |
| OLED Display      | 1      | 128x32 I2C display module           | Status display |
| Vibration Motor   | 1         | 3V DC mini motor                    | Haptic feedback |
| CD-42 Board       | 1         | Power breakout / interface board    | Power management |
| Battery           | 1         | 3.7V Li-ion or LiPo                 | Power source |
| Switch            | 1         | SPST toggle or push button          | Power control |
| Jumper Wires      | ~20       | Male-to-male                        | Connections |
| Breadboard        | 1         | 830 tie-points                      | Prototyping base |
